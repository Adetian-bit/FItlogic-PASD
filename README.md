# FitLogic - Sistem Rekomendasi Diet Berbasis Data

FitLogic adalah aplikasi web berbasis data yang membantu pengguna menghitung kebutuhan kalori harian, BMI, BMR, TDEE, serta memberikan rekomendasi makronutrisi dan estimasi progres diet dalam bentuk dashboard interaktif.

Aplikasi ini dibuat untuk tujuan edukasi dan implementasi konsep Object Oriented Programming (OOP) pada JavaScript dengan dukungan Python untuk proses export ke HTML.

---

## 🚀 Demo
Aplikasi dapat dijalankan melalui GitHub Pages atau membuka file `index.html` secara langsung di browser.

---

## 📌 Fitur Utama

- Perhitungan BMI (Body Mass Index)
- Perhitungan BMR (Basal Metabolic Rate)
- Perhitungan TDEE (Total Daily Energy Expenditure)
- Target kalori berdasarkan tujuan (cutting, bulking, maintain)
- Rekomendasi makronutrisi (protein, karbohidrat, lemak)
- Diet Match Score
- Estimasi waktu pencapaian target berat badan
- Dashboard interaktif:
  - Progress berat badan
  - Timeline program
  - Checklist harian
  - Jurnal berat badan
- Glosarium istilah gizi

---

## 🧠 Teknologi yang Digunakan

- HTML5
- CSS3
- JavaScript (Object Oriented Programming)
- Python (untuk generate dan export file HTML)
- LocalStorage (penyimpanan data lokal)

---

## 📊 Struktur Sistem

Aplikasi menggunakan pendekatan OOP pada JavaScript dengan class utama:

- `User`
- `CalorieCalculator`
- `MacroRecommendation`
- `DietProgress`
- `DietMatchScore`
- `StorageManager`
- `Dashboard`
- `FitLogicApp`
